Proyecto Técnico Industrial

Automatización Sostenible de un Invernadero

Abril 2026 Universidad Carlos III de Madrid (UC3M) Agricultura Inteligente

Resumen del Proyecto

Este proyecto se centra en el diseño e implementación de un sistema de control climático y riego automatizado para un invernadero de alta tecnología. El objetivo principal es garantizar la sostenibilidad y eficiencia en el cultivo, optimizando el uso de recursos hídricos y energéticos mediante la monitorización en tiempo real de las condiciones ambientales.

Detalles Técnicos

El sistema se ha diseñado bajo una arquitectura jerárquica industrial robusta e incluye los siguientes componentes clave:

  • Controlador Lógico Programable (PLC): Uso de Siemens SIMATIC S7-1200 para la adquisición de datos de sensores y control de actuadores, programado bajo normativas industriales.
  • Interfaces HMI y SCADA: Desarrollo de un sistema de supervisión y control de datos que permite a los operadores visualizar variables como temperatura, humedad (aire y suelo), niveles de CO₂ y radiación solar.
  • Instrumentación Industrial: Selección, dimensionamiento y acondicionamiento eléctrico de sensores meteorológicos, válvulas proporcionales y variadores de frecuencia, comunicados vía PROFINET y lazos de corriente 4-20mA.
  • Seguridad y Normativa: Cumplimiento de la normativa nacional y europea para cuadros eléctricos e instalaciones, integrando elementos críticos de protección y sistemas SAI para evitar la pérdida de datos.
